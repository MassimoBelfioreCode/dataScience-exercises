# data-manipulation-exercises
Repo with my data analysis exercises with numpy, pandas, matplotlib.


**Datasets Used**

This repository includes data analysis exercises performed on the following datasets:

    Titanic - Machine Learning from Disaster

        Source: Kaggle / agconti GitHub

        Key Skills: Data cleaning, handling missing values (NaN), categorical encoding (mapping 'male'/'female' to 'M'/'F'), and advanced grouping with pd.crosstab.

        Dataset: https://raw.githubusercontent.com/agconti/kaggle-titanic/master/data/train.csv

    Google Play Store Apps

        Source: googleplaystore.csv (Kaggle)

        Key Skills: Data type conversion (parsing strings to numeric), data cleaning, and statistical aggregation to find the average number of reviews per app category.

    Students Admission Dataset

        Source: University of Catania (DMI)

        Key Skills: File I/O operations, exploring correlations between GRE scores, GPA, and university admission rates.

        Dataset: http://iplab.dmi.unict.it/furnari/downloads/students.csv

    Synthetic Data & Custom Dataframes

        Focus: Core Pandas and NumPy logic.

        Key Skills: Creating Series from dictionaries, merging multiple dataframes (Inner/Natural Join), using groupby for financial data (e.g., employee incomes by company), and vectorization with NumPy.
